
pour lancer le serveur de bd
----------------------------
source start.sh

pour créer le schema de bd (qui est dans db.sql) (ne faite pas ca, laissez , spring va creer lui même le schema)
------------------------------------------------
source populate.sh

Pour lancer la console permettant de taper des requetes SQL
-----------------------------------------------------------
source console.sh

le driver a inclure dans votre projet (dans lib)
-------------------------------------------------------
hsqldb/lib/hsqldb.jar
